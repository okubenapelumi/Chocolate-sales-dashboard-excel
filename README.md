# 📊 Chocolate Sales Dashboard (Excel Project)

## 🔍 Overview

This project analyzes chocolate sales data to uncover insights into revenue performance, customer purchasing behavior, and product trends. The goal was to transform raw data into a clear, interactive dashboard for decision-making.

---

## 🛠️ Tools & Techniques

* Microsoft Excel
* Data Cleaning & Preparation
* Pivot Tables & Pivot Charts

---

## 📈 Dashboard Features

* Sales performance by country
* Purchase level segmentation (Low, Medium, High)
* Product-level analysis
* Customer classification (e.g., retailer vs wholesaler)
* Clean and structured visual layout for easy interpretation

---

## 🎯 What This Project Demonstrates

* Ability to clean and structure raw data
* Strong understanding of Excel functions and logic
* Data storytelling through dashboards
* Turning data into actionable insights
